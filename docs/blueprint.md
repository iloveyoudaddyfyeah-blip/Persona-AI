# **App Name**: PersonaCraft AI

## Core Features:

- Photo Upload & Analysis: Users upload a photo of a person. AI analyzes facial features, attire, and background to infer personality traits, interests, and background details.
- Personality Profile Generation: Generative AI crafts a detailed personality profile based on the photo analysis, with a maximum text limit of 10,000 characters. This includes a biography, strengths, weaknesses, hobbies, and potential motivations.
- Interactive Chat: Users can engage in a chat-like interaction with the AI, asking questions about the generated character's life, opinions, and experiences. The AI tool will make choices to either weave some information into the AI generated answer. The AI has a memory store of 10,000 characters.
- Profile Customization: Users can manually adjust the generated profile to fine-tune details or correct inaccuracies. They can modify traits, background information, and motivations.
- Character Repository: Ability to store and manage multiple created characters.

## Style Guidelines:

- Primary color: Warm, inviting light blue (#ADD8E6), reminiscent of Waterfall in Undertale, evokes feelings of earthiness, comfort, and approachability
- Background color: Very light, desaturated gray (#F0F8FF) reminiscent of the CORE.
- Accent color: Complementary muted red (#FF6961), hints at determination.
- Body and headline font: 'Determination' pixel font, with a literary, vintage feel.
- Clean, minimalist pixel art icons representing different profile aspects (traits, background, etc.).
- Card-based layout for character profiles. Clean and intuitive interface for chat interaction, inspired by Undertale's dialogue boxes.
- Subtle transitions and animations when displaying profile details or new chat messages. Animations based on Undertale's battle screen.
